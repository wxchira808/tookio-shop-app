## ğŸ†˜ Help & Support

### Common Issues:

**"Database connection error"**
- Check `DATABASE_URL` in `apps/web/.env`
- Make sure schema.sql is loaded in Neon

**"Cannot connect to localhost"**
- Make sure `npm run dev` is running
- Try http://localhost:3000 in browser

**"Module not found"**
- Run `npm install` in both apps
- Delete `node_modules` and reinstall

**Mobile app blank screen**
- Check console for errors
- Press 'r' to reload
- Try `npx expo start -c` to clear cache
